# 🕯️ לעילוי — אפליקציית תפילות, השכבות והילולות

אפליקציית ווב אינטראקטיבית (React + TypeScript + Vite) לפי [תוכנית העבודה](../תוכנית-עבודה.md).

## הרצה

```bash
npm install
npm run dev      # שרת פיתוח — http://localhost:5199
npm run build    # בניית ייצור לתיקיית dist
```

## מה יש בפנים

| לשונית | יכולת |
|---|---|
| **היום** | תאריך עברי, פרשה, מה אומרים היום (משיב הרוח/מוריד הטל, ברך עלינו, יעלה ויבוא, הלל, תחנון), עומר, זמני היום והשבת לפי עיר |
| **תפילות** | בנק תפילות וסגולות עם חיפוש עברי (בלי ניקוד), קטגוריות, הגדלת גופן, הדפסה |
| **אזכרה** | מחולל סידור אזכרה: תהילים, פסוקי קי"ט לפי אותיות השם + נשמ"ה, השכבה (ספרדי) / א-ל מלא רחמים (אשכנזי) עם השם משולב, קדיש — וייצוא PDF |
| **יקיריי** | ניהול נפטרים, חישוב אזכרה שנתי לפי ההלכה (כולל אדר בשנה מעוברת), שבעה/שלושים/י"א חודשי קדיש |
| **הילולות** | לוח הילולות צדיקים שמתעדכן אוטומטית כל שנה, עם חיפוש |

## ארכיטקטורה

- `src/lib/calendar.ts` — מנוע הלוח (עטיפה ל-@hebcal/core)
- `src/lib/yahrzeit.ts` — חישובי אזכרה ואבלות. **שימו לב:** `getYahrzeit` של hebcal משנה את האובייקט שמועבר אליו — מעבירים עותק טרי בכל קריאה
- `src/lib/hebrew.ts` — נירמול עברית: חיפוש, אותיות השם, הסרת טעמים והחלפת שם ה' (`sanitizeSacred`)
- `src/data/` — תוכן: תפילות, נוסחי השכבה, הילולות, תהילים (מ-Sefaria)
- `scripts/fetch-tehillim.mjs` — הורדת פרקי תהילים מ-Sefaria API (מריצים שוב כדי להוסיף פרקים)

## ⚠️ חשוב

הנוסחים והתאריכים **טעונים בדיקה רבנית** לפני שימוש רשמי או פרסום. הטקסטים המקראיים מקורם ב-Sefaria (רישיון חופשי); נוסחי התפילה הם נוסחים מסורתיים בנחלת הכלל.
